Sobre mim.

Comecei meus estudos no ínicio de 2021 desenvolvendo pequenos algoritmos em C para trabalhos na faculdade onde estudo, até chegar em conceitos de Estruturas de Dados e algoritmos de ordenação mais complexos (como Bubblesort, Mergesort, etc) utilizando essa mesma linguagem. Desde então, venho aprimorando meus conhecimentos em diferentes linguagens e tecnologias, tais como C#, Java, Python e JavaScript.

Recentemente estou usando a linguagem Java para produzir sistemas com Spring, além de também estudar React, Node.JS e .NET sempre que tenho a oportunidade.
